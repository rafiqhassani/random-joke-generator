# Random Joke Generator

A simple web app that displays random jokes at the click of a button. Perfect for a quick laugh!

##Features
. Shown a random joke when you click a button.
. Easy-to-use, responsive interface.
. Can be extended to fetch jokes from ApIs.
. Simple JavaScript implementation for learning purpose.

## Live Demo

You can see the live demo link of this project here: [Random Joke Generator](https://rafiqhassani.github.io/random-joke-generator/)

## How to Use

1. open the live demo link.
2. Click the button to generate a new random joke.
3. Enjoy and share with your friends!

## Technologies Used

- HTML
- CSS
- JavaScript
